<h1 align="center">JS Arrays</h1>

<p>
  Raccolta di esercizi in JavaScript dedicati alla manipolazione di array, con particolare attenzione
  all’uso del ciclo <code>for</code> e alle basi della logica iterativa.
</p>

## Obiettivo

- Prendere confidenza con gli array in JavaScript:
  - lettura degli elementi;
  - aggiunta e modifica dei valori;
  - ricerca e confronto di elementi.
- Risolvere le varie task partendo da codice già impostato.
- Utilizzare il ciclo <code>for</code> come strumento principale per:
  - scorrere gli array;
  - applicare condizioni;
  - costruire risultati passo dopo passo.
- Verificare i risultati usando la console del browser.

## Contenuto degli script

Gli script contengono piccoli esercizi che lavorano su array, ad esempio:

- gestione di liste di valori (stringhe o numeri);
- combinazione di più array;
- controlli su elementi presenti o meno;
- semplici trasformazioni dei dati senza usare metodi avanzati come
  <code>map</code>, <code>filter</code> o <code>includes</code>.

L’output degli esercizi viene ispezionato tramite <code>console.log()</code> e DevTools.

## Tecnologie utilizzate

- JavaScript
